# Common instructions

These documents are reusable building blocks for built-in module skills. They are
resources, not standalone skills: read them when a skill links to them, then continue
the active module-scoped `run`.

- [Grilling](okh://instructions/grilling.md) — resolve one user decision at a time.
- [Ingest](okh://instructions/ingest.md) — turn source documents into routed, cited candidates.
- [OKF writer](okh://instructions/okf/writer.md) — author grounded OKF bundles.
- [OKF format](okh://instructions/okf/format.md) — OKF conformance reference.
